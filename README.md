# Higgs Loop-Induced Processes

## h_to_gamma_Z

This folder contains the routines used to compute the decay width of the Higgs boson into a Z boson and a photon, both within the Standard Model (SM) and the MSSM + RNS framework.

The routines can be straightforwardly adapted to compute the loop-induced Higgs decay widths into two photons and two gluons.

## loop_induced

This folder contains Mathematica routines and data for plotting Higgs decay widths and signal strengths associated with loop-induced Higgs decays into gauge bosons.
