# Data and Codes for arXiv:2604.22115

This folder contains the data and codes required to reproduce the results presented in the paper arXiv:2604.22115.

## Contents

1. The programs with the `.wls` extension generate the corresponding plots. Additional details and instructions can be found inside each code.

2. The compressed files `outputs.zip` and `outputs2.zip` contain the data required to generate the plots. Please extract these folders in the same location where the codes are stored, and replace the variable `Dir` inside the codes with the path corresponding to that location.
